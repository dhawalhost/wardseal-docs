# About WardSeal

WardSeal is an **Open Core** Identity & Access Management (IAM) platform designed for modern cloud infrastructure. Our mission is to provide enterprise-grade security tools that are accessible, transparent, and easy to integrate.

## The Open Core Model

WardSeal is built on a foundation of transparency and community. We believe that the core building blocks of identity should be open for everyone to audit, secure, and contribute to.

- **Community Edition**: Our fully-featured, open-source core allows developers to self-host their own IAM infrastructure on Kubernetes or Docker. It includes everything you need for OIDC, SAML, and robust user management.
- **WardSeal Cloud**: For organizations that need a managed solution, we offer a fully hosted SaaS platform. WardSeal Cloud provides high availability, automated scaling, and advanced security features tailored for enterprise compliance.

## Why WardSeal?

- **Auditable Security**: Since the core is open-source, you can verify exactly how your data is handled.
- **Protocol-First**: Native support for modern standards like OIDC, SAML, SCIM, and WebAuthn.
- **Developer Experience**: Extensive APIs and a beautiful administration console designed to get you up and running in minutes.
